# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-26

- Initial release of the Infomaniak node pack.
- One node per product generated from OpenAPI specs (Resource/Operation UI with dedicated fields).
- OAuth2 and API Key credentials.
- Pagination support (limit/skip and page/per_page) with retries on rate limiting.
- Product-specific icons with fallback icon.
